Content blocks

Ari comes with many content blocks (or designed blocks) which helps you to present your data/information in pages, blog post or any places you want to.

Alert​

Type declaration
type AlertProps = {
  alertType?: AlertTagColor

  className?: string
  hasShadow?: boolean

type FaqsBlockProps = {

  answer: string

Map​

Map component allows you to show a map and you can also add many custom marker. When hovering over a marker, it information would be display on the top right of the map.

Type declaration
type MapProps = {
  markers: TMarker[]
  /**
   * zoom is from `4` to `18`
   */
  defaultZoom?: number
  defaultCenter?: Point
  hasPadding?: boolean
  alignment?: "left" | "right" | "expanded"
}

export type TMarker = {
  width?: number
  color?: string
  anchor: Point
  slug?: string
  title?: string
  description?: string
  /**
   * title and description would be displayed
   * as if the marker is on hover if `isDefault` is true
   * Tip: only set 1 marker with `isDefault = true`
   */
  isDefault?: boolean
}

/**
