# Content blocks

Aris comes with many content blocks *(or designed blocks)* which helps you to present your data/information in pages, blog post or any places you want to.

## Alert

<details>

<summary>Type declaration</summary>

```ts
type AlertProps = {
  alertType?: AlertTagColor
  alignment?: "left" | "right"
  className?: string
  hasShadow?: boolean
  children: React.ReactNode
  /**
   * make your alert looks `art` when skew to the left or to the right
   */
  art?: "right" | "left"
}

type AlertTagColor = number | "gray" | "blue" | "red" | "yellow" | "green" | "random" | "info" | "success" | "warning" | "error"
```

</details>

## Image

<details>

```ts
type ImageProps = JSX.IntrinsicElements["img"] & {
  srcList: ImgSrcSet,
  rounded?: "sm" | "md" | "lg" | "xl" | "2xl"
  wfull?: boolean,
  hfull?: boolean
}

type ImgSrcSet = {
  mobile: string
  desktop?: string
  description?: string
}
```

</details>

<Tabs>

## Map

Map component allows you to show a map and you can also add many custom marker. When hovering over a marker, it information would be display on the top right of the map.

<details>

<summary>Type declaration</summary>

```ts
type MapProps = {
  markers: TMarker[]
  /**
   * zoom is from `4` to `18`
   */
  defaultZoom?: number
  defaultCenter?: Point
  hasPadding?: boolean
  alignment?: "left" | "right" | "expanded"
}

export type TMarker = {
  width?: number
  color?: string
  anchor: Point
  slug?: string
  title?: string
  description?: string
  /**
   * title and description would be displayed
   * as if the marker is on hover if `isDefault` is true
   * Tip: only set 1 marker with `isDefault = true`
   */
  isDefault?: boolean
}

/**
 * [latitude, longitude]
 */
export type Point = [number, number]
```

</details>

<Tabs>
<TabItem value="Usage">

```tsx
import Map from "~/components/Map"

<Map
  alignment="expanded"
  defaultCenter={[10.8161863, 106.7782831]}
  defaultZoom={16}
  hasPadding={true}
  markers={myMarkers}
/>
```

</TabItem>
<TabItem value="Data">

```ts
export const myMarkers: TMarker[] = [
    {
      anchor: [10.8119658,106.7759439],
      slug: "/blog/labore-nisi-cillum-voluptate-consequat-reprehenderit-laborum-deserunt-exercitation",
```
